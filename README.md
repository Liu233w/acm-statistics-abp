# acm-statistics-abp
[新版 NWPU-ACM 查询系统](https://github.com/Liu233w/acm-statistics)后端

## 运行环境
- [.Net Core Sdk](https://www.microsoft.com/net/download) 2.0 或以上 （只在编译时需要）
- MySQL 5.7 或以上
- Windows 或 Linux 系统均可
